# OwnerView (오너뷰) — 서비스 상세 가이드

> **"대표님, 회사 상황이 한눈에 보이시나요?"**
> 회계사무소 없이도, ERP 없이도 — 회사의 돈, 사람, 서류를 한 곳에서.

---

## 오너뷰란?

**오너뷰**는 중소기업/스타트업 대표를 위한 **올인원 회사운영 OS**입니다.

엑셀로 관리하던 자금현황, 홈택스에서 따로 보던 세금계산서, 플렉스로 처리하던 급여, 카카오톡으로 하던 업무 소통 — 이 모든 것을 **하나의 화면**에서 처리합니다.

### 핵심 가치
- **5-in-1 통합**: 재무 + 세무 + 문서/계약 + 인사/급여 + 협업 채팅
- **생존 지표**: 현금 잔고, 런웨이, 번율을 실시간으로
- **자동화**: 거래 분류, 세금계산서, 급여 계산, 결재 — 수동 입력 최소화
- **경쟁사 대비 72% 절감**: 월 9,900원부터

## 사용자별 가이드

### 1. 대표 (Owner)

> **"내 회사가 몇 개월 버틸 수 있는지, 지금 바로 알고 싶다"**

#### 대시보드 — 생존 지표 6-Pack
로그인하면 가장 먼저 보이는 화면입니다.

| 지표 | 설명 | 왜 중요한가? |
|------|------|------------|
| **현금 잔고** | 모든 계좌 합산 잔액 | 지금 당장 쓸 수 있는 돈 |
| **런웨이** | 현금 소진까지 남은 개월 수 | 3개월 이하면 위험 신호 |
| **월 번율** | 매월 나가는 고정비 총액 | 줄여야 런웨이가 늘어남 |
| **미수금** | 아직 받지 못한 매출 | 현금흐름의 숨은 변수 |
| **미수금 30일+** | 30일 이상 연체된 미수금 | 회수 불능 위험 |
| **승인 대기** | 결제/결재 미처리 건수 | 병목 = 돈이 묶임 |

- **위험 신호 자동 감지**: 런웨이 3개월 이하, 미수금 급증, 고정비 초과 시 빨간 경고
- **현금흐름 예측**: 향후 30일/90일 현금 흐름 시뮬레이션
- **매출 차트**: 월별 매출 추이 + 드릴다운 (큰 숫자 → 세부 내역까지)

#### 승인센터 — 원클릭 결재
대시보드에서 바로 처리 가능합니다:
- 결제 승인 (급여, 고정비, 개별 결제)
- 경비 청구 승인
- 휴가 신청 승인
- 문서 승인
- 결재 요청 승인
- 세금계산서 발행 승인

**한 곳에서 모든 승인을 처리**합니다. 각각 다른 메뉴에 들어갈 필요 없습니다.

#### 재무 관리
- **결제 큐**: 나가야 할 돈 목록 → 승인 → 실행 (5가지 채널)
- **거래내역**: 은행/카드 CSV 업로드 → 자동 분류 → 딜/거래처 연결
- **세금계산서**: 매출/매입 관리 + 홈택스 엑셀 임포트 + VAT 미리보기
- **대출 관리**: 대출별 상환 스케줄 + 잔액 추적
- **매칭 엔진**: 거래 ↔ 세금계산서 ↔ 주문서 3-Way 자동 매칭

#### 전체 접근 권한
대표는 **모든 페이지**에 접근 가능합니다:
- 딜/프로젝트 관리
- 거래처 CRM
- 문서/계약서 생성·서명
- 인사/급여 전체
- 자산 금고
- 설정 (팀 관리, 은행 계좌, 결재 정책)
- SaaS 관리 패널

### 2. 관리자 (Admin)

> **"대표님 대신 실무를 처리하는 경영지원/재무 담당자"**

#### 접근 가능한 기능
- **대시보드**: 전체 현황 조회 (승인 권한 없음, 대표에게 전달)
- **딜 관리**: 프로젝트 생성/수정, 마일스톤 관리
- **거래처 CRM**: 거래처 등록/수정, 연락처 관리
- **결제 관리**: 결제 큐 등록, 지출결의서 작성, 급여 일괄 생성
- **세금계산서**: 발행/수정, 홈택스 엑셀 임포트
- **거래내역**: 은행/카드 거래 업로드, 자동 분류 규칙 설정
- **문서/계약**: 문서 작성, 템플릿 관리, 서명 요청
- **결재**: 결재 요청 생성, 정책 관리
- **인사/급여**: 직원 등록/초대, 급여 관리, 계약서 발송, 근태/휴가 승인
- **채팅**: 팀 채널 + 딜 채널 참여
- **설정**: 회사 정보, 은행 계좌, 팀 초대

#### 관리자만의 업무
- 직원 초대 및 권한 설정
- 경비 청구 1차 승인
- 휴가 신청 승인
- 결재 정책 설정 (금액별 승인자 지정)
- 월말 결산 체크리스트 관리

### 3. 직원 (Employee)

> **"출퇴근, 휴가, 경비청구만 간편하게"**

#### 접근 가능한 기능

**근태 관리**
- 출근/퇴근 버튼 원클릭
- 주간/월간 근무시간 자동 계산
- 초과근무 시간 추적

**휴가 관리**
- 잔여 연차 확인 (시간/반일/일 단위)
- 휴가 신청 (캘린더에서 날짜 선택)
- 신청 현황 추적 (대기중 → 승인 → 사용완료)
- 유형별 신청: 연차, 반차(오전/오후), 시간차, 병가, 경조사, 공가

**경비 청구**
- 지출 내역 등록 (금액, 카테고리, 영수증 첨부)
- 승인 상태 추적
- 카테고리: 교통비, 식비, 접대비, 소모품, 출장비, 교육비

**증명서 발급**
- 재직증명서 PDF 즉시 발급
- 경력증명서 PDF 즉시 발급
- 발급 이력 자동 기록

**협업**
- 할당된 딜/프로젝트 조회
- 팀 채팅 참여
- 문서 조회 (권한 범위 내)

---

### 4. 파트너 (외부 거래처)

> **"초대받은 딜에 대해서만 소통"**

#### 접근 가능한 기능
- **딜 조회**: 초대받은 딜의 진행 상황 확인
- **채팅**: 해당 딜 채팅방 참여 (파일 공유 포함)
- **문서**: 공유받은 견적서/계약서 확인
- **서명**: 계약서 전자서명

파트너는 회사 내부 재무/인사 데이터에는 접근할 수 없습니다.

## 주요 기능 상세

### A. 프로젝트/딜 관리

모든 비즈니스를 **딜(Deal)** 단위로 관리합니다.

**딜 = 수익이 발생하는 모든 프로젝트**
- 클라이언트 프로젝트
- 정부 사업 수주
- 상품 납품
- 서비스 계약

**기능:**
- 트리 구조 (상위 딜 → 하위 항목, 무한 계층)
- 수익/비용 실시간 계산 + 마진율 색상 코딩
- 마일스톤 관리 (D-day 자동 계산)
- 팀원 역할 할당
- 거래처 자동 연결
- 딜 → 견적서 → 계약서 → 세금계산서 자동 파이프라인

---

### B. 결제 자동화 (5가지 채널)

| 채널 | 설명 | 흐름 |
|------|------|------|
| **결제 큐** | 개별 결제 건 | 등록 → 대표 승인 → 실행 |
| **지출결의/품의** | 직원이 비용 신청 | 신청 → 관리자 승인 → 결제 큐 자동 추가 |
| **급여 일괄** | 월급 일괄 처리 | 직원 선택 → 4대보험 자동 계산 → 일괄 승인 |
| **고정비 일괄** | 월 고정 비용 | 반복결제 목록 → 월별 자동 생성 |
| **반복결제** | 구독/임대료 등 | 거래내역에서 패턴 감지 → 자동 등록 제안 |

**4대보험 자동 계산:**
- 국민연금: 기준소득월액 × 9% (사업주/근로자 각 4.5%)
- 건강보험: 보수월액 × 7.09% (사업주/근로자 각 3.545%)
- 고용보험: 월급여 × 1.85% (사업주 1%, 근로자 0.85%)
- 소득세: 간이세액표 기준 자동 산출

---

### C. 세무 관리

**세금계산서 관리**
- 매출/매입 세금계산서 등록
- 홈택스 엑셀 파일 일괄 임포트
- 월별 VAT 미리보기 (납부세액 예상)
- 3-Way 매칭: 주문서 + 거래내역 + 세금계산서 자동 대조

**거래내역 자동 분류**
- 은행/카드 CSV 업로드 → 거래처/비용 유형 자동 분류
- 분류 규칙 학습 (같은 거래처 패턴 기억)
- 미분류 거래 하이라이트 → 수동 매핑 → 규칙 자동 저장

---

### D. 문서/계약 관리

**문서 작성**
- 빈 문서 또는 템플릿에서 시작
- Markdown 에디터
- 버전 관리 (수정 이력 추적)
- 딜 데이터 자동 채우기 (견적서 → 금액/거래처 자동 입력)

**전자서명**
- 이메일로 서명 링크 발송
- 3가지 서명 방식: 그리기 / 타이핑 / 이미지 업로드
- 직인(회사 도장) 자동 오버레이
- 서명 완료 시 PDF 잠금 + 이력 기록

**계약 패키지**
- 여러 서류를 한 패키지로 묶어 발송 (예: 연봉계약서 + NDA + 개인정보동의서)
- 서명 진행 상태 실시간 추적
- 모든 서명 완료 시 급여 자동 반영

**문서 공유**
- 토큰 기반 외부 공유 (로그인 불필요)
- 공유 대상의 피드백 수집 (승인/반려/코멘트)
- 공유 만료일 설정

**기본 템플릿 5종:**
1. 연봉계약서
2. 비밀유지서약서 (NDA)
3. 겸업금지서약서
4. 개인정보 동의서
5. 포괄임금 근로계약서

### E. 인사/급여 (HR)

**8개 탭 통합 관리:**

| 탭 | 기능 | 자동화 |
|---|------|--------|
| **인력관리** | 직원 정보 + 연봉 + 퇴직충당금 | 퇴직충당금 자동 적립 |
| **급여이력** | 급여 변동 기록 | 계약서 서명 시 자동 반영 |
| **급여 명세** | 월별 급여 명세서 | 4대보험 자동 계산 |
| **계약서** | 계약 패키지 발송/서명 추적 | 서명 완료 → 급여 자동 세팅 |
| **경비청구** | 지출결의서 신청/승인 | 승인 시 결제 큐 자동 추가 |
| **근태** | 출퇴근 기록 | 주간 근무시간 자동 계산 |
| **휴가** | 연차 신청/승인 | 연차 자동 발생 + 미사용 촉진 알림 |
| **증명서** | 재직/경력증명서 발급 | PDF 즉시 생성 |

**연차 자동 관리:**
- 입사일 기준 연차 자동 발생
- 시간/반일/일 단위 사용
- 11개월 미사용 시 자동 사용 촉진 알림
- 연말 미사용 연차 소멸 관리

### F. 협업 채팅

- **딜별 채팅방**: 프로젝트마다 전용 채팅방 자동 생성
- **팀 채팅방**: 부서/팀별 소통 채널
- **실시간 메시지**: WebSocket 기반 즉시 전달
- **파일 공유**: 드래그 앤 드롭 업로드
- **@멘션**: 특정 팀원 태그 + 알림
- **반응**: 이모지 반응
- **검색**: 메시지 내용 검색
- **외부 초대**: 거래처/파트너를 딜 채팅방에 초대

### G. 자산 금고 (Vault)

회사의 모든 자산을 한 곳에서 관리합니다.

| 유형 | 예시 |
|------|------|
| **계정** | SaaS 구독 (Slack, Notion 등), 결제수단, 로그인정보 |
| **유형 자산** | 노트북, 사무기기, 차량 |
| **무형 자산** | 소프트웨어 라이선스, 특허, 도메인 |
| **문서** | 사업자등록증, 인증서, 보험증서 |

**자동 발견**: 은행/카드 거래에서 반복 결제 패턴을 감지하여 "이 서비스도 등록하시겠습니까?" 제안

### H. 데이터 가져오기 (Import Hub)

**6가지 파일 자동 감지:**

| 파일 유형 | 설명 | 결과 |
|----------|------|------|
| **은행 CSV** | 입출금 거래내역 | → 거래내역 자동 등록 + 분류 |
| **카드 CSV** | 법인카드 승인내역 | → 카드 거래 등록 + 거래처 매핑 |
| **홈택스 엑셀** | 세금계산서 합계표 | → 세금계산서 자동 등록 |
| **CEO 보고서** | 자금예산 현황 | → 현금 스냅샷 업데이트 |
| **Flex HR 엑셀** | 직원/급여 데이터 | → 직원 정보 + 급여 자동 세팅 |
| **인수인계 문서** | 프로젝트/거래처/고정비 | → 딜 + 거래처 + 비용 자동 생성 |

**사용법**: 파일을 드래그 앤 드롭 → 자동 감지 → 미리보기 → 확인 → 자동 등록

### I. 결재 시스템

**다단계 결재 워크플로우:**
1. 직원이 요청 생성 (지출/휴가/문서 등)
2. 정책에 따라 승인자 자동 지정 (금액/유형별)
3. 1차 승인 → 2차 승인 → ... → 최종 승인
4. 승인 타임라인 시각화
5. 반려 시 사유 기록 + 재제출 가능

**결재 정책 설정:**
- 금액 기준: 10만원 이하 = 팀장, 100만원 이상 = 대표
- 유형 기준: 경비 = 관리자, 계약 = 대표
- 건너뛰기 가능 (긴급 시)

---

### J. 온보딩 (처음 시작하기)

**6단계 셋업 마법사:**
1. 회사 정보 입력 (회사명, 사업자번호)
2. 은행 계좌 등록
3. 법인카드 등록
4. 홈택스 연동 (엑셀 임포트)
5. 직원 등록/초대
6. 완료 (대시보드로 이동)

또는 **엑셀 한 장으로 전체 세팅**:
- 인수인계/현황판 엑셀 업로드 → 프로젝트, 거래처, 고정비, 직원 일괄 등록
